🏥 BIOME Care — Sistema de Turnos y Planes de Salud

BIOME Care es una aplicación web desarrollada para optimizar la gestión de atención médica, 
permitiendo a los usuarios solicitar turnos, comparar planes de salud y a los 
administradores gestionar las solicitudes de forma clara, organizada y profesional.

🚀 Funcionalidades principales

👩‍⚕️ Pacientes
Solicitud de turnos online
Selección de especialidades médicas
Validación de formularios
Prevención de turnos duplicados
Simulación y recomendación de planes de salud

🏥 Administración (panel profesional)
Visualización de turnos registrados
Eliminación de turnos
Interfaz clara y responsive
Persistencia de datos con LocalStorage

💳 Planes de salud
Comparación de coberturas
Recomendación automática según perfil del usuario
Simulación de contratación
